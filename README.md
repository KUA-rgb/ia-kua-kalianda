# IA KUA Kalianda

Integrated Application untuk Kantor Urusan Agama (KUA) Kalianda

## Deskripsi Proyek

Aplikasi web terintegrasi untuk mengelola data pernikahan, perceraian, dan rujuk di KUA Kalianda.

## Tech Stack

- **Frontend**: React 18 + Vite
- **Backend**: Python FastAPI
- **Database**: PostgreSQL
- **DevOps**: Docker & Docker Compose

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Docker Compose
```bash
docker-compose up -d
```
